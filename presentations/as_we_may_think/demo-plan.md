# Demo Plan

## 1. Basic agent usage
- Talk to an agent, show tool calls in action
- Visible loop: read → think → act → fix

## 2. Skills
- Show structured prompts and persistent memory
- Run a skill, show the agent reading/writing to files

## 3. Subagent swarms
- Parallel agents coordinating work
- Show the live agent tree in the TUI

## 4. Collaboration
- Shared memory, shared agents

## Demo project idea
- **Conway's Game of Life as a Slidev component**
- Ask the agent to build it live — Vue component with canvas, themed to the presentation
- Runs inside the slide deck itself — visual, self-contained, satisfying
- Good error-and-fix potential (canvas rendering, game logic)
- Could use the swarm demo to add features in parallel (e.g. click-to-toggle, colour themes, speed controls)
